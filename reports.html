Reports

Reports can extract data from the API for automated generation of text, figures (maps, time series, etc) and tables in a variety of formats (html, pdf, docx). This is especially made possible with Rmarkdown (see 1 min video: What is R Markdown? on Vimeo).

This web page reports.html was rendered from reports.Rmd which can also be rendered in these other formats:

  • Adobe: reports.pdf
  • Word: reports.docx

str_trunc <- function(s, nchar_max = 20){
  if (nchar(s) > nchar_max){
    s <- str_sub(projects$name[1], end=nchar_max) %>% 
      paste0("...")
  }
  return(s)
}

kbl <- function(tbl){
  if (fmt != "html"){
    out <- kable(tbl) 
  } else {
    out <- kable(tbl) %>%
      kable_styling(bootstrap_options = c("striped", "hover"))
  }
  out
}
